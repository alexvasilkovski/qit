# qit
A quirky CLI for interacting with git.
## Installation
***make sure you have ~/.bin in your $PATH***
```bash
git clone https://github.com/alexvasilkovski/qit
cd qit
chmod +x qit
mv qit ~/.bin
cd ..
rm -r qit/
```
## Usage

```bash
$ qit - Coming Soon...

```
